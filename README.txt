安裝指南

安裝前準備

"環境檔案" 資料夾包含所有必要檔案：
    CUDA (exe 檔)
    cuDNN
    Anaconda (exe 檔)
    Visual Studio Code (exe 檔)

安裝步驟：

1.安裝 CUDA 和 cuDNN
    使用 GPU 功能需要安裝 CUDA 和 cuDNN。
    安裝 CUDA、cuDNN 教學 : https://medium.com/ching-i/win10-%E5%AE%89%E8%A3%9D-cuda-cudnn-%E6%95%99%E5%AD%B8-c617b3b76deb

2.安裝 Anaconda 和 Visual Studio Code
    建議使用 Anaconda 和 Visual Studio Code，將exe檔執行即可。

3.執行 train_ui.py
    即可開始使用。
